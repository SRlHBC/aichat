# 🐱 AI 猫娘聊天室
AI 聊天助手 (aichat)  |  GitHub: github.com/SRlHBC/aichat
- 基于 Streamlit 构建交互式 Web 界面，集成 OpenAI API 实现多轮对话
- 实现对话历史保存/加载功能，提升用户体验
- 使用 Git 进行版本管理，项目结构清晰，提供完整 README 和运行说明
- 技术栈：Python, Streamlit, REST API, Git
一个基于 Streamlit + DeepSeek API 的可爱猫娘聊天应用。你可以自定义猫娘的名字和性格，与她进行多轮对话，并支持会话的保存、加载与删除。

![Streamlit](https://img.shields.io/badge/Streamlit-1.35%2B-red)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ 功能特性

- 🎭 **自定义猫娘**：自由设置猫娘的名字和性格（如“傲娇”、“温柔”、“爱撒娇”等）。
- 💬 **实时对话**：基于 DeepSeek API 的流式回复，体验打字机效果。
- 📂 **会话管理**：
  - 自动保存聊天记录到本地 JSON 文件。
  - 支持创建多个会话，随时切换历史记录。
  - 可删除不需要的会话。
- 🐾 **角色扮演提示**：内置详细的猫娘设定，让对话更生动可爱（喵～）。
- 🖥️ **侧边栏控制面板**：新建会话、会话列表、角色设置一目了然。
